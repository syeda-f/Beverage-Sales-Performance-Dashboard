# Beverage-Sales-Performance-Dashboard
This project presents an interactive Power BI dashboard for analyzing beverage sales performance across four markets (USA, UAE, India, UK). The dashboard tracks key business metrics such as total sales ($89M), profit, cost, customer performance, country-wise sales, and sales trends over time.

# Key Features
 + Sales, profit, and cost KPI tracking
 + Country-wise sales analysis (USA, UAE, India, UK)
 + Customer and customer segment performance analysis
 + Year-over-year sales and profit trends
 + Top 10 customers by revenue
 + Profit breakdown by product and region
 + Interactive filters and visualizations

# Insights
 + Identified top-performing countries and customers
 + Analyzed the most profitable customer segments
 + Tracked yearly and quarterly sales and profit trends
 + Replaced manual weekly Excel reporting with a real-time dashboard
 + Supported data-driven business decision-making
